# トピックス差し替え方法

トップページのトピックスは `topics-data.js` だけを編集します。

## 1件の形

```js
{
  status: "show",
  date: "2026.06.22",
  label: "お知らせ",
  title: "タイトル",
  body: "本文",
  link: "./guide.html#hours",
  linkText: "診療時間を見る",
}
```

## よく使う変更

- 表示する：`status: "show"`
- 非表示にする：`status: "hide"`
- 画像を入れる：必要な時だけ `image: "./assets/illust-画像名.png"` を追加
- リンクなしにする：`link: ""`、`linkText: ""`
- 並び順を変える：項目の順番を入れ替える
- 削除する：`{ ... }` の1ブロックを消す

## 注意

本文を正確に保つため、トピックスの文字は画像ではなくHTML文字として表示します。画像を入れる場合も、本文は必ずHTML文字として残します。
